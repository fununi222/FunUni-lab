Glossary

ブログ内で登場する専門用語や略語を、記事を読み進めやすい粒度で整理したページです。

OpenClaw

OpenClaw

ブラウザ操作、シェル実行、検索などを組み合わせてタスクを進める AI エージェント実行環境です。 このブログでは、作業を分解して進める「実務向けの操作レイヤー」として扱っています。

AI Agent

AIエージェント

文章を返すだけでなく、指示に応じてツールを使いながら実際の作業まで進める AI のことです。

Orchestrator

オーケストレーター

複数の処理やツールの流れを調整し、どの順番で何を実行するかを管理する役割です。 記事内では、OpenClaw が各専門スキルへ仕事を振り分ける司令塔として登場します。

Wrapper

ラッパー

複雑な処理を直接見せず、呼び出しやすい形に包んで使えるようにする仕組みです。

PoC

PoC

Proof of Concept の略で、「その案や技術が本当に成立するか」を小さく試す検証のことです。

OAuth

OAuth

外部サービスに対して、パスワードを直接渡さずにアクセス権を認可するための仕組みです。

Google Calendar API

Google Calendar API

Google カレンダーの予定取得や作成を、プログラムから行うための公式 API です。

vSphere

vSphere

VMware が提供する仮想化基盤の総称で、仮想マシンやホストをまとめて管理できます。

CBT

CBT

Changed Block Tracking の略で、前回バックアップ以降に変化したブロックだけを追跡する仕組みです。

用語は随時追加されます。不明な点があれば研究ログを参照してください。
